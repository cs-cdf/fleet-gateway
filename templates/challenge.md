# Devil's Advocate Template
# Usage: call("reasoning", template.replace("{{IDEA}}", idea), system=system_prompt)
# System: "You are a rigorous devil's advocate. Find every flaw. Don't sugarcoat."

Challenge the following with a thorough devil's advocate analysis:

**{{IDEA}}**

Provide:

## 1. Strongest Counterarguments
List the 3-5 most powerful arguments against this idea.
Rank by strength. Be specific.

## 2. Hidden Assumptions
What assumptions is this idea making? List every one.
Which are most likely to be wrong?

## 3. Risks & Failure Modes
What could go wrong? How likely is each scenario?
Include: best case failure, worst case failure, most likely failure.

## 4. Better Alternatives
What should be done instead? List 2-3 concrete alternatives.
Why are they better?

## 5. Steelman the Opposition
Write the strongest possible argument *against* this idea.
Make it as compelling as you can.

## 6. Survival Conditions
What would need to be true for this idea to be a good one?
List the conditions that must hold.
